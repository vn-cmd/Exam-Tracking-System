University Exam Registration System
Overview

This project is a system designed for universities or academies to manage exam registrations efficiently. It involves three main user roles: Student, Professor, and Administrator.

Features

Student:

Registers for exams related to subjects they are enrolled in

Views available exam dates for their subjects

Can sign up for and cancel exam registrations

Professor:

Manages exam dates for the subjects they teach

Approves or rejects student exam registrations (e.g., in cases of academic misconduct)

Publishes exam results

Uploads study materials for each exam

Views and prints lists of successfully registered students

Administrator:

Has full system oversight and management privileges

Assigns professors, students, and subjects within the system

Accesses all features available to professors

Monitors the overall system

Intended Use

This system is aimed at academic institutions to streamline the exam registration process, improve communication between students and professors, and simplify administrative tasks related to exams
